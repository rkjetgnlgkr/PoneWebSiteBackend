⏺ /init 是 Claude Code 的初始化指令，主要功能是：

  自動分析你的專案並產生 CLAUDE.md 檔案，這個檔案包含：
  - 專案的 build / test 指令
  - 架構概覽
  - 程式碼規範與慣例
  - 常用工作流程

  用途：讓 Claude 在每次新對話開始時都能自動讀取這份文件，不需要每次重新解釋專案結構，保持跨 session 的一致性。

  注意事項：
  - 如果 CLAUDE.md 已存在，/init 不會覆蓋，而是建議改進內容
  - 這個檔案可以納入版本控制（git commit），讓整個團隊共用
  - 盡量保持在 200 行以內，避免消耗太多 token

  如果你在這個 Spring Boot 專案下執行 /init，Claude 會自動偵測到 Maven、Spring Boot 設定，並把常用指令（如 mvn
  spring-boot:run）記錄進去，之後就不需要每次重新說明。

     os` + `portfolio_images`。
  43
  44 **檔案上傳**：存放於本地 `./uploads/`，透過 `WebConfig` 映射為靜態資源，對外路徑為 `/api/files/{filename}`，此路徑不需 J
     WT。

  47
  48 - MySQL：`localhost:3306`，資料庫名 `pone_website`，帳號 `root`

⏺ JwtFilter.doFilterInternal 的執行時機與原理

  什麼時候執行？

  每一個 HTTP 請求進來時都會執行，在到達 Controller 之前。

  呼叫鏈（由外到內）

  HTTP Request
      ↓
  Servlet Container (Tomcat)
      ↓
  CorsFilter（先處理跨域）
      ↓
  JwtFilter.doFilterInternal  ← 在這裡驗證 Token
      ↓
  DispatcherServlet
      ↓
  Controller

  機制原理

  1. 繼承 OncePerRequestFilter

  public class JwtFilter extends OncePerRequestFilter

  OncePerRequestFilter 是 Spring 提供的抽象類別，保證同一個請求只執行一次（避免 forward/include 時重複執行）。它實作了 Java
  Servlet 規範的 javax.servlet.Filter 介面。

  2. Spring 自動註冊

  @Component  // ← 關鍵
  public class JwtFilter extends OncePerRequestFilter

  加上 @Component 後，Spring Boot 啟動時會自動偵測所有實作 Filter 的 Bean，並透過 FilterRegistrationBean 將它註冊到 Tomcat 的
  Filter Chain 中。

  3. Filter Chain 機制

  filterChain.doFilter(request, response);  // 放行，交給下一個 Filter 或 Controller

  Tomcat 維護一條 Filter Chain，每個 Filter 決定：
  - 呼叫 filterChain.doFilter() → 放行，繼續往下走
  - 不呼叫，直接寫 response → 攔截，請求到此為止

  在你的 JwtFilter 中：
  - Token 驗證通過 → 呼叫 filterChain.doFilter() 放行
  - Token 無效或未提供 → 呼叫 writeUnauthorized() 直接回傳 401，不放行

  整體流程圖

  Tomcat 收到請求
      │
      ▼
  JwtFilter.doFilterInternal()
      ├─ 路徑是 /auth/login 或 /api/files/**?
      │       ↓ Yes → filterChain.doFilter() → 放行
      │
      ├─ 有 Authorization Bearer header?
      │       ↓ No → writeUnauthorized(401)，結束
      │
      ├─ Token 有效?
      │       ↓ No → writeUnauthorized(401)，結束
      │
      └─ 全部通過 → filterChain.doFilter() → 到達 Controller
